# SkillMatch JS

## Sobre o projeto

O SkillMatch JS é um simulador simples de compatibilidade entre um candidato e vagas de front-end júnior.

O projeto compara as habilidades do candidato com os requisitos das vagas e mostra:

- percentual de compatibilidade;
- habilidades encontradas;
- habilidades faltantes;
- vaga mais compatível;
- recomendação de estudo.

### Como a internet funciona?

A internet é uma grande rede mundial de computadores e servidores conectados entre si, permitindo a troca de informações e arquivos.

Cada dispositivo conectado à internet possui um endereço IP, que funciona como uma identificação na rede e permite o envio e recebimento de dados entre diferentes dispositivos e servidores.

A comunicação entre navegador e servidor pode ser protegida pelo protocolo HTTPS, que criptografa os dados transmitidos, aumentando a segurança das informações.
